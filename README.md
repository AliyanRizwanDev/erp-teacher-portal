# Education Portal Frontend - Portfolio Project

**⚠️ DEMO PROJECT**: This is a portfolio/demonstration project built as a code sample. It uses mock data and is not connected to any live systems.

Production-style faculty operations portal built with React, TypeScript, Vite, and Tailwind CSS.

## Overview

This project demonstrates a modular, production-quality frontend architecture for an education management system. It showcases:

- Professional authentication flow (demo mode - no real credentials required)
- Responsive dashboard with profile management and activity feeds
- Multiple feature modules: attendance tracking, results management, timetable operations, leave/loan applications, complaints and suggestions submission
- Fully responsive mobile-first design for desktop and tablet breakpoints
- Reusable component library with form controls, modals, and data tables
- Comprehensive pagination and filtering capabilities

- Login system (demo credentials: any email/password accepted)

- Leave application workflow
- Loan application workflow
- Complaints and suggestions submission

## Tech Stack

- React 19
- TypeScript 5
- Vite 6
- Tailwind CSS 4
- React Router
- React Hook Form
- Radix UI primitives

## Project Structure

- src/components: shared and feature UI components
- src/pages: route-level page modules
- src/config: route definitions and page routing map
- src/hooks: reusable state and pagination hooks
- src/utils: static demo data and helper collections
- public/images: static assets used by UI surfaces

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run local development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```
